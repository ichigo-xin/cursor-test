# 网站目录结构

## 前端目录结构
```
frontend/
├── public/                 # 静态资源
├── src/
│   ├── assets/            # 资源文件
│   │   ├── images/       # 图片资源
│   │   └── styles/       # 样式文件
│   ├── components/        # 公共组件
│   │   ├── common/       # 通用组件
│   │   └── layout/       # 布局组件
│   ├── views/            # 页面组件
│   │   ├── FileConvert/  # 文件转换
│   │   ├── ImageCompress/# 图片压缩
│   │   ├── ImageCrop/    # 图片裁剪
│   │   ├── ImageConvert/ # 图片格式转换
│   │   └── ImageWatermark/# 图片水印
│   ├── router/           # 路由配置
│   ├── store/            # 状态管理
│   ├── utils/            # 工具函数
│   ├── api/              # API接口
│   ├── App.vue          # 根组件
│   └── main.js          # 入口文件
├── package.json          # 项目配置
└── vite.config.js       # Vite配置
```

## 后端目录结构
```
backend/
├── src/
│   ├── config/          # 配置文件
│   ├── controllers/     # 控制器
│   │   ├── fileController.js    # 文件处理
│   │   └── imageController.js   # 图片处理
│   ├── services/        # 服务层
│   │   ├── fileService.js       # 文件服务
│   │   └── imageService.js      # 图片服务
│   ├── routes/          # 路由
│   ├── utils/           # 工具函数
│   ├── middleware/      # 中间件
│   └── app.js          # 应用入口
├── uploads/            # 上传文件临时目录
├── package.json        # 项目配置
└── .env               # 环境变量
```
